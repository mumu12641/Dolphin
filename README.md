## :warning: Warning

This project is for **educational purposes only**. This repository does not provide any script services or automation tools.

## :book: Features

- :calendar: **Smart Reservation**: Easily book venues with a few taps.
- :pushpin: **History & Quick Start**: Automatically saves your reservation configurations (as seen in the "Empty History" state) for one-click re-booking.
- :art: **Material Design 3**: Customize the application theme to your liking. 
- :moon: **Dark Mode**: Fully supported dark theme for comfortable night usage.
